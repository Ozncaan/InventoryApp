# InventoryApp

Android + Kotlin ile yazılmış stok yönetim uygulaması.

## Özellikler

- Barkod tarama (CameraX + ML Kit)
- Ürün ekleme, silme, düzenleme
- Arama ve filtreleme
- Kritik stok uyarısı

## Kurulum

1. Android Studio ile projeyi aç  
2. `local.properties` dosyasına `sdk.dir` ekle  
3. Build ve Run
